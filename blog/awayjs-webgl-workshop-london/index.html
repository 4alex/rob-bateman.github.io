AwayJS @ WebGL Workshop London

Rob Bateman

Later today I'll be heading down to the WebGL workshop meetup in London to give a short pres on the progress of AwayJS. For those of you who don't know, AwayJS is the latest and greatest library development from The Away Foundation, and something we have been working on now for the best part of 2 years.

I'll be talking about the installation process we now use, taking advantage of Node's package manager npm for deployment and the Typescript language for coding. I'll also demo some of the new workflow tools we've been busy developing to make designing content for use in AwayJS a breeze.

Details of the event can be found here, talks start at around 6.30. Some great graphics guys included in the lineup, should be a good evening!
